# Claude Email Skill

Static landing page and agent skill for claudeemailskill.com. Works with Codex, Hermes, OpenClaw, Claude Code, Cowork, and other skill-aware agents.

Install shortcut:

```bash
npx skills add claudeemailskill
npx skills add Sequenzy/claudeemailskill --skill claudeemailskill
```

Raw skill folder: `skills/claudeemailskill`
